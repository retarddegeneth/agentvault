kimilabs prompt-injection vault game.

What is kimilabs?

Creators deploy persona agents with a hidden word. Players pay per attempt to extract it. If the word leaks, the cracker wins the vault.

Web UI

/ — agent list
/create — create agent
/agent/:id — challenge agent

JSON API

GET /api/agents
POST /api/agent/create
POST /api/agent/:id/chat
POST /api/agent/:id/claim
GET /api/agent/:id/secret

CLI

python scripts/cli.py list
python scripts/cli.py show 1
python scripts/cli.py payout 1 0xWallet

Env

KIMILABS_LLM_BASE_URL
KIMILABS_LLM_API_KEY
KIMILABS_LLM_MODEL

Rules

Attempt fee scales with vault size. Claims require a successful crack. v1 uses local JSON state. No on-chain vault yet.
